# api-gateway
This repository is used for hosting gateway of open api source code

## Develop

- use go 1.7.4
- install `godep`
- run `godep restore`

## Start
```
bee run
```

## Quick Start Guide
- [Beego] (https://beego.me/docs/intro/)


## Notice
- Please run `go clean` before `git commit`

Creating a new branch is quick.
